## Assignment Overview

This assignment demonstrates Oracle Pluggable Database (PDB) administration, including creating a PDB, creating a user, granting privileges, deleting a temporary PDB, and accessing Oracle Enterprise Manager.

## Oracle Environment

- Oracle Version: Oracle Database 21c Express Edition
- Operating System: Windows
- Tools Used: SQL*Plus and Oracle Enterprise Manager (OEM)

## Task Documentation

- Created a new user inside the PDB.
- Granted the required privileges.
- Logged in using the created user.
- Created and deleted a temporary PDB.
- Accessed Oracle Enterprise Manager (OEM).
